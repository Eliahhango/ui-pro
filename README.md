# ElitechWiz Website

Premium dark cyber-tech website built with Next.js App Router, React, Tailwind CSS, and Framer Motion.

## Important Content Rule

This project is configured to render only verified business facts.

- Contact info: edit app/business.ts -> contacts
- Services: edit app/business.ts -> services

No fake testimonials, no fake metrics, no blog/shop/news sections.

## Local Development

```bash
npm install
npm run dev
```

## Deploy

Deploy directly on Vercel from your GitHub repository.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion
